## Installation


1. Clonez ce depot :
   ```bash
   git clone <url-du-depot>
   cd gomoku
   ```

2. Crez un environnement virtuel (recommand) :
   ```bash
   python -m venv venv
   source venv/bin/activate  # Sur Windows: venv\Scripts\activate
   ```

3. Installez les dependances :
   ```bash
   pip install -r requirements.txt
   ```
ou 
```bash
   ./scripts.sh
```

### Lancement manuel

```bash
python gomoku.py
```
